# cleanDataProject
This repository contains work for the Get and Clean data project from the Coursera Data Science coursework sponsored by John Hopkins University

The file stucture is as follows:

  1. **run_analysis.R** - this script contains code to transform raw datasets contained within repository to the output tidy dataset, tidyData.csv. Our script uses the `data.table` framework to read, transform, and output our tidy data set. 
  2. **tidyData.csv** - the output dataset from run_analysis.R. 
  3. **getdata_projectfiles_UCI HAR Dataset.zip** - contains the raw download from the assignment home page on coursera.
  4. **UCI HAR Dataset** - extract files from the zip file taken from the assignment home page. These folders contain the raw input data for run_analysis.R.
